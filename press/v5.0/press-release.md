## Noisen 5.0 Brings Live Multiplayer to the Meditative Sound App

*No accounts. No setup. Shared sound in seconds.*

Noisen, the browser-based ambient sound generator loved by meditators, musicians and sleep hackers, today releases version 5.0 — its biggest update yet. The centerpiece is **real-time multiplayer**: anyone can now share a live sound session with a link, letting multiple devices shape the same sonic canvas simultaneously.

---

### What is Noisen?

Noisen is a minimalist Progressive Web App (PWA) for generative ambient sound. Users place nodes on an infinite canvas — each node's horizontal position sets its frequency, vertical position its filter brightness. Nodes interact through simulated gravity. No menus to navigate, no account to create. Works offline, installable on any device, plays through silent mode on iOS.

---

### What's New in 5.0

#### Multiplayer Sessions
- Create a session, copy the link, share it — collaborators join your canvas in real time
- Everything syncs: filter, volume, step sequences, orbits, FX sends, node type changes, mute
- Session survives page reload — reconnects automatically as host or guest
- Works on mobile, desktop, and across different networks

#### Smarter Drums
- New drum nodes now arrive with a sensible euclidean rhythm pattern — no more silent nodes on first placement
- Comets (orbiting gravity bodies on the canvas) now **trigger drum nodes on flyby** — orbit a comet through your percussion kit for live, unpredictable rhythm
- All drum parameters sync across multiplayer in real time

#### Full Parameter Sync
- Global sliders (Volume, Gravity, Tone, Spread) broadcast to all session participants
- Orbits (LFO modulation) sync: add, remove, adjust rate/depth/target in real time
- Node type switching, mute, and all synthesis parameters propagate instantly
